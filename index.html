CertPlus Mandados

Guia Rápido para Oficiais de Justiça

Bem-vindo ao CertPlus Mandados! Este sistema foi criado para automatizar a leitura de mandados em PDF (Saj e Eproc) usando Inteligência Artificial, enviando-os para uma planilha no Google Drive.

O aplicativo (leve e acessível via URL no celular ou PC) extrai os dados, cria um “Card” para cada mandado e permite: consultar/enviar o PDF, registrar atos no momento do cumprimento, gerar certidões automáticas e digitalizar mandados assinados. Tudo organizado em uma pasta restrita a você!

Começar a Configuração

Tempo estimado: menos de 5 minutos ⏱️

PRÉ-REQUISITO: Liberação de Acesso

O aplicativo possui uma trava de segurança. Antes de começar, certifique-se de que você enviou o seu E-mail do Google (Gmail ou conta Google vinculada) para o administrador do sistema liberar o seu acesso. Sem isso, o aplicativo de celular será bloqueado.

Gerar Chave API

A inteligência artificial precisa de uma chave de acesso para ler seus PDFs.

  • Acesse o Google AI Studio. (Faça login com sua conta Google).
  • No menu lateral esquerdo, procure por "Get API key".
  • Clique no botão azul "Create API key" (se for a primeira vez, escolha "Create API key in new project").
  • Uma janela aparecerá com letras e números. Clique em "Copy" e guarde essa chave para o próximo passo!

Preparando os PDFs

Vá até o seu Google Drive, no menu lateral esquerdo clique em "Meu Drive".

Lá você vai notar que o sistema criou uma pasta chamada "CertPlus - Mandados Digitalizados", onde vai estar a planilha denominada "Cópia de [MODE]CertPlus - Matriz Oficial v2".

ATENÇÃO

Nesta mesma pasta "CertPlus - Mandados Digitalizados" coloque os arquivos em PDF dos Mandados que fez carga, do sistema "SAJ" e "EPROC".

Cada arquivo em PDF tem que corresponder a APENAS UM Mandado, não importando a quantidade de páginas existentes no arquivo (cópias do Mandado e seus anexos).

11

A Mágica no Aplicativo

Após colocar os PDFs na pasta, retorne ao aplicativo e clique no botão de atualizar ou dê F5:

Você verá que o sistema vai carregar uma lista de "Cards", sendo que cada um representa um Mandado. A planilha também vai receber informações sobre cada Mandado (uma linha para cada), que ficará para sempre no seu "Drive", com todas as informações pertinentes e um sistema de consulta muito simples.

COMO USAR NO DIA A DIA

Quando for diligenciar para cumprir algum Mandado, antes, no CELULAR, procure ele na lista de "Cards" por meio da barra de pesquisa/busca. Ao clicar vai ver informações pertinentes, como os dados do Mandado, link para o "Google Maps", link para abrir o PDF do Mandado, e outras possibilidades que vão facilitar seu dia a dia, inclusive:

  • Emissão de certidão (diversos modelos prontos e preenchidos automaticamente) para copiar e colar no Saj/Eproc.
  • Autos diversos com preenchimento facilitado e assinatura na tela do celular.
  • Possibilidade de tirar fotos do Mandado assinado ou de objetos, guardadas e organizadas por número de Mandado numa pasta gerada pelo sistema.
  • Entre outras possibilidades, como o registro da data e do horário da diligência de forma automática, se você registrar o ato no celular logo em seguida do cumprimento, etc.

Dicas de Ouro e Solução de Problemas

"Nenhum PDF novo encontrado"

Significa que o robô olhou na pasta da planilha e não achou arquivos .pdf. Verifique se você colocou os PDFs exatamente na mesma pasta onde a planilha está salva.

App em looping infinito

Acontece se tiver mais de uma conta Google logada no navegador do celular (ex: pessoal e tribunal), causando conflito. Solução: Abra o link em uma Aba Anônima e logue apenas com a conta autorizada.

Ícone na Tela Inicial do Celular

No Chrome (Android) ou Safari (iPhone), abra o aplicativo, clique nos 3 pontinhos do navegador (ou botão compartilhar) e role a lista para encontrar a opção "Adicionar à Tela Inicial". Ele vira um ícone de app normal!
